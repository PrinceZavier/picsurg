# PicSurg

A machine learning application that automatically identifies operative/surgical photos in your iPhone camera roll and securely transfers them to a HIPAA-compliant, password-protected folder.

## Features

- **ML-Powered Recognition**: Automatically detects surgical and operative photographs
- **Secure Storage**: Moves identified images to a password-protected folder
- **HIPAA Compliance**: Ensures protected health information (PHI) is handled according to HIPAA security requirements
- **Camera Roll Cleanup**: Removes sensitive medical images from your main photo library
